1. Main Page 操作說明

1.1 事前準備

請先準備好一個 .csv 或是 .xlsx 檔案，其中

  • 第一欄（the first column）必須是產品名稱，其他欄位沒有規定。
  • 第一列（the first row）請給予各欄位標題。像是第一欄可以命名為 prd name 或是 Product Name。

注意事項：請注意上傳 .csv 最終下載的就是 .csv 檔， .xlsx 亦同。

注意事項：商品名稱越具體，匹配結果越準確。例如「iPhone 16 128GB」比「iPhone」更容易找到精確匹配的商品。

1.2 平台選擇說明

本系統支援多平台商品搜尋，您可以選擇以下平台：

🛒 Momo購物網

台灣最大電商平台，商品種類豐富

🌏 Coupang

韓國領先電商平台，適合跨境商品研究

🔄 兩個平台都選

同時搜尋兩個平台，會需要花兩倍的時間

1.3 預設選擇

預設選擇：系統預設選擇 Coupang 平台，您可以根據需求調整。

1.4 操作步驟

1選擇要搜尋的電商平台（可單選或多選）

  • 勾選 "Momo" 以搜尋 Momo購物網
  • 勾選 "Coupang" 以搜尋 Coupang
  • 可同時勾選兩個平台進行比較

2上傳檔案

點擊 "Upload" 按鈕以上傳 .csv 或 .xlsx 檔案

3自動處理

系統會自動開始爬蟲處理，無需額外點擊

4監控與下載

等到 "Progress" 欄位大於或等於 0%時，就可以進行底下兩種操作

  • 點擊 "view" 在網站上觀察目前爬蟲輸出檔案狀況，方便檢查。請注意每爬 10 筆資料檔案才會更新一次
  • 點擊 "download" 按鈕就可以將爬到現在的輸出檔案下載下來。如果有要將連結丟到另外一個系統就可以事先下載不用等

1.6 結果說明

輸出檔案會根據您選擇的平台包含對應的商品名稱和連結。沒有對應連結的可能情況如下：

  • No search result = 爬蟲在該平台沒有搜尋結果。您可以自行複製商品名稱到平台官網測試確認
  • No corresponding item = 雖然有搜尋結果但是 AI 判斷搜尋結果中沒有匹配的產品

2. 熱門商品趨勢功能使用說明

📈 2.1 操作步驟

全新的熱門商品趨勢分析功能，讓您輕鬆掌握 Coupang 平台的市場動態：

1進入趨勢分析頁面

點擊導航選單中的「Coupang Trending」連結

2選擇商品類別

從下拉選單中選擇您感興趣的商品類別：

Phone Only
手機類商品
Phone Accessories
手機配件
Computer Laptop
筆記型電腦
Computer Accessories
電腦配件
Game
遊戲相關商品
Home Electronic
家電產品

3選擇查詢日期

使用日期選擇器挑選您要查看的日期，系統會顯示該日期的熱門商品資料

4瀏覽趨勢數據

系統會載入對應的熱門商品清單，表格包含以下功能：

  • 搜尋功能：可在表格中搜尋特定商品
  • 排序功能：點擊欄位標題進行排序
  • 分頁瀏覽：大量數據自動分頁顯示
  • 商品連結：直接點擊商品名稱前往 Coupang 商品頁面

5下載數據

提供兩種下載選項：

  • Download Selected Category：下載當前選擇類別的 CSV 檔案
  • Download All (ZIP)：下載該日期所有類別的 ZIP 壓縮檔

小提示：頁面會顯示趨勢資料的建立時間，方便您掌握資訊的時效性。
